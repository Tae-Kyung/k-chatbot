K-Student AI Guide - Widget Demo

이 페이지는 챗봇 위젯이 외부 사이트에 삽입된 모습을 보여주는 데모입니다.

위젯 삽입 방법:

아래 코드를 웹페이지의 </body> 태그 앞에 삽입하세요.

<script
  src="https://your-domain.com/widget.js"
  data-university="a1b2c3d4-e5f6-7890-abcd-ef1234567890"
  data-lang="ko"
></script>

속성 설명

  • data-university: 대학교 ID (필수)
  • data-lang: 기본 언어 (ko, en, zh, vi, mn, km)
  • data-base-url: API 서버 주소 (생략 시 스크립트 호스트 사용)

지원 대학교

  • 충북대학교: a1b2c3d4-e5f6-7890-abcd-ef1234567890
  • 한국교통대학교: b2c3d4e5-f6a7-8901-bcde-f12345678901
  • 한국교원대학교: c3d4e5f6-a7b8-9012-cdef-123456789012
